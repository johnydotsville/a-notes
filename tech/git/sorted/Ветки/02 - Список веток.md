# Список веток

## Только локальные ветки

```
git branch
```

* Показывает все ветки, которые у нас есть локально в этом репозитории.
* Звездочкой отмечает текущую ветку.

## Все ветки

```
git branch -a
```

* Показывает все ветки, которые есть в этом репозитории, и локальные, и remote.
* Звездочкой отмечает текущую ветку.
