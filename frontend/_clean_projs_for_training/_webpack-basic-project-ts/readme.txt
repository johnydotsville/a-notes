* Что это?

Это базовый webpack-проект для тренировок с typescript.

* Что здесь есть?

Подключен TS, dev-север и плагин, подключающий бандл к странице.

* Как пользоваться?

1. Через консоль перейти в корневую директорию проекта (там где файл package.json).
2. Установить пакеты командой npm i
3. Запустить проект на dev-сервере командой npm run start
4. Страница автоматически откроется в браузере, выбранном по умолчанию.
   Адрес будет http://localhost:3000/
5. Для остановки dev-сервера нажать Ctrl + C

Можно обойтись без dev-сервера. Тогда надо билдить проект вручную
командой npm run build:dev (или build:prod) и открывать html-страницу,
появившуюся в папке dist, в браузере.
