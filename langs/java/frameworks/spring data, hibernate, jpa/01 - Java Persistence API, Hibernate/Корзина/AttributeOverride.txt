С помощью этой аннотации можно переопределить только аннотацию @Column, 
т.е. по сути только переопределить имя столбца и еще пару мелочей.
С помощью нее нельзя переопределить, например, имя последоватьности,
которая используется для получения id.


Кроме того, AttributeOverride, похоже, может использоваться только для
переопределения полей в классе, помеченным @MappedSuperclass. При использовании
других стратегий работы с наследованием, AttributeOverride уже может не работать.
